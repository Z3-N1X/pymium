# pymium
Easiest way to create GUI with python supporting Windows, Mac, Linux.
This project is also avaible at [Pypi](https://pypi.org/p/pymium).

# instalation
```python
pip install pymium
```
Use this at the top of your python file
```python
import pymium
```
